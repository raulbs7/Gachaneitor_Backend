Como crear una receta

Es importante saber que tipo de pasos se deben seguir para pasar una receta a este repositorio.

Modo de introducción

Existen dos modos para introducir las recetas:

  • Por medio un archivo subido.
  • En la propia página, por medio de un área de texto.

En caso de que se introduzcan ambas opciones, se le dará mayor prioridad al archivo subido.

Como escribir la receta

El lenguaje utilizado para introducir recetas es el que ya se ha definido en la documentación del proyecto.

Esto es un lenguaje específico del dominio en el que la receta se define entre llaves. En ella se separa el tiempo, el nombre (y descripción), ingredientes y pasos, los cuales cada uno de estos van entre llaves también.
